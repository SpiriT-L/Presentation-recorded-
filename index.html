CSS Grid Layout

CSS Grid is a modern and most powerful layout technique which is available in CSS that allows designing a web page using a grid-based layout system.

CSS grid is a two-dimensional system as compared to other design elements like using flexbox. It helps web developers in the alignment of web elements according to needs and the elements can be divided into different regions with relationships such as size, layer, position, etc. Doing some changes in the Grid layout system wonâ€™t affect the other elements. CSS grid layout system is different from CSS frameworks available and it allows web developers more freedom in terms of design.

Grid Container: display property

For making the grid we define a grid-container which is a parent and all the elements in it are grid-child.

The parent element here is defined with display : grid property. This will set the grid to the block level. This property is must while defining grid layout. All the child elements in grid container will automatically be treated as grid childs.

Grid Container: grid-template-columns

This property will divide the grid layout into based out of columns. The number of values supplied will decide a number of columns.
